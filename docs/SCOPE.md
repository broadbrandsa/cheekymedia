# Scope

Fixed-fee website rebuild for Cheeky Media — **R30,000**.

## Pages included

| Page          | Purpose                                                        |
| ------------- | -------------------------------------------------------------- |
| Home          | Hero, positioning, services overview, featured work, CTA.      |
| About Us      | Story, approach, team.                                         |
| Services      | Digital Media, Brand Integration, Design, Development, Branding.|
| Portfolio     | Filterable work grid (Commercials, Corporate, Digital, Film, TV).|
| Contact       | Enquiry form + contact details + map/location.                 |

## Sections included (componentised)

- Sticky responsive navigation + mobile menu
- Hero with brand story messaging
- Services grid
- Featured / portfolio showcase
- Process / approach
- Team
- Testimonials / social proof (if supplied)
- Contact / lead capture
- Footer with socials

## Included in the R30,000

- Custom responsive design (mobile-first) on Next.js + Tailwind.
- Up to 5 core page templates as above.
- Reusable component library (shadcn/ui based).
- On-page SEO basics (meta, semantic markup, sitemap, OG tags).
- Contact form with email/lead routing.
- Analytics setup (GA4 + Vercel Analytics).
- Vercel deployment + domain connection.
- One round of revisions per page after first review.

## We can take care of (if needed)

- **Copywriting** — Cheeky Media can supply the words, or Broadbrand can write the
  site copy on request.
- **Stock photography** — where Cheeky Media has no imagery, Broadbrand will source
  and licence suitable stock photography.

## Cheeky Media provides

- **CI / brand guidelines** — colours, typography, tone and usage rules.
- **Logo files** — all formats (vector preferred: SVG / EPS / PDF, plus PNG).
- **Work-done assets** — portfolio pieces: project stills, video, client names and
  any details to feature (Commercials, Corporate, Digital, Film, TV).

## Excluded (quoted separately)

- E-commerce / payments.
- Custom CMS build or headless CMS integration beyond basic content editing.
- Original photo / video shoots (stock sourcing is covered above).
- Bespoke third-party / API integrations.
- Ongoing maintenance, hosting management, or retainer support.
- Brand identity redesign (logo/visual system) — design *applies* existing brand.

## Launch milestones

1. **Discovery & sign-off** — kick-off, confirm scope, assets, sitemap.
2. **Design direction** — homepage concept + component styling approved.
3. **Build** — all pages developed against approved direction.
4. **Content load & review** — real content in, client review round.
5. **QA & launch** — cross-device QA, analytics live, deploy to production.
