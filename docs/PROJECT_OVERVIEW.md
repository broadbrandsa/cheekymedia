# Project Overview

## What this is

This repository is a **standalone proposal website** that pitches a full website
rebuild for **Cheeky Media** (https://cheekymediaworld.com). It is not the rebuilt
product site itself — it is the sales artifact used to win that engagement.

The proposal presents the scope, approach, timeline, deliverables and pricing for
rebuilding Cheeky Media's web presence at a fixed fee of **R30,000**.

## About the client

- **Client:** Cheeky Media — a Johannesburg-based creative & media agency.
- **Positioning:** "Culture engineers, audience whisperers, and storytellers with purpose."
- **Tagline:** "Your Brand is more than a Logo. It's a STORY, and we're here to tell it."
- **Services:** Digital Media, Brand Integration, Design, Development, Branding.
- **Current site:** Built by Plus27Digital; dated structure and presentation.
- **Contact:** admin@cheekymedia.co.za · 011 258 4465 · 43 Central St, Houghton Estate, Johannesburg, 2198.

## Target audience

- Cheeky Media decision-makers (founders / directors) evaluating the rebuild.
- Secondary: anyone on their side reviewing scope and cost before sign-off.

## Goal

Communicate, clearly and persuasively, **what they get, when, and for how much** —
and make it trivially easy to approve and move forward.

## Deployment environment

- **Framework:** Next.js (App Router) + TypeScript + Tailwind CSS v4.
- **UI primitives:** shadcn/ui.
- **Hosting:** Vercel.
- **Package manager:** pnpm.
